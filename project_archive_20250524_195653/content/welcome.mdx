## üíª Code Example

Here's a complete example showing how to authenticate and send your first message:

```javascript
// 1. Authenticate and get token
const authResponse = await fetch('https://api.telegent.com/v1.0/oauth2/tokens', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    AccountKey: 'DID-your-account-key',
    AccountSecret: 'your-account-secret',
    ApiEndpoint: 'https://api.telegent.com/v1.0/message/outbound'
  })
});

const { token } = await authResponse.json();

// 2. Send a message
const messageResponse = await fetch('https://api.telegent.com/v1.0/message/outbound', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    To: [{ Number: '+1234567890' }],
    From: '+1987654321',
    Body: 'Hello from Telegent mPaaS!',
    Owner: '+1987654321'
  })
});

const result = await messageResponse.json();
console.log('Message sent:', result.MessageId);
```
