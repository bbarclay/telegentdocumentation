## Base URL

All API requests should be made to the following base URL:

```
https://api.telegent.com/
```

## API Versioning

Telegent APIs are versioned. The current version is `v1.0`. Always include the version in your API endpoints:

```
https://api.telegent.com/v1.0/
```

## Authentication

All API requests require authentication using either:

- **Pop-Token**: A session token obtained through the OAuth2 authentication endpoint
- **API Key**: For select endpoints that accept direct API key authentication

Learn more in the [Authentication](/api-reference/authentication) section.

## Response Format

All API responses are returned in JSON format with PascalCase property names.

## Error Handling

The API returns appropriate HTTP status codes along with error details in the response body:

```json
{
  "Type": "Error type",
  "Title": "Error title",
  "Status": 400,
  "Errors": {
    "Field": ["Error message"]
  },
  "TraceId": "Unique error trace ID"
}
```
