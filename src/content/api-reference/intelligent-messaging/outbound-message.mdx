## Message Types

The API supports the following message types:

- **SMS**: Standard text message
- **MMS**: Multimedia message that can include images, audio, or video

## Including Media in Messages

To include media in your MMS messages, you have two options:

1. **Media URLs**: Provide URLs to publicly accessible media files
   ```json
   "MmsMedia": [
     {"MediaUrl": "https://example.com/image.jpg"}
   ]
   ```

2. **Base64 Encoded Media**: Embed base64 encoded media directly in the request
   ```json
   "MmsBase64": [
     {
       "Base64": "base64EncodedString",
       "BaseType": "image/jpeg"
     }
   ]
   ```
