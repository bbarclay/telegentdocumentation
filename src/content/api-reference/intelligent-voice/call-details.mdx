---\ntitle: Call Details\n---\n\n## Call Details\n\nThis endpoint allows you to retrieve detailed information about a specific voice call using its unique `CallId`.\n\n### Endpoint\n\n`/voice/calls/{CallId}`\n\n**Method:** `GET`\n\n### Path Parameters\n\n| Parameter | Type   | Description                       | Required |\n|-----------|--------|-----------------------------------|----------|\
| `CallId`  | string | The unique identifier of the call. | Yes      |\n\n### Response Body (200 OK)\n\n| Parameter      | Type   | Description                                  |\n|----------------|--------|----------------------------------------------|\
| `CallId`       | string | Unique identifier for the call.              |\n| `From`         | string | The originating phone number.                |\n| `To`           | string | The destination phone number.                |\
| `Direction`    | string | The direction of the call (`inbound` or `outbound`). |\n| `Status`       | string | The current status of the call (e.g., `initiated`, `in-progress`, `completed`, `failed`, `busy`, `no-answer`). |\
| `StartTime`    | string | Timestamp when the call started.             |\
| `EndTime`      | string | Timestamp when the call ended (if applicable). |\
| `Duration`     | integer| Duration of the call in seconds (if applicable). |\
| `ErrorCode`    | string | Error code if the call failed.               |\
| `ErrorMessage` | string | Error message if the call failed.            |\
| `ClientState`  | string | Custom data associated with the call.        |\n\n```json\n{\n  \"CallId\": \"call_12345abcde\",\n  \"From\": \"+15551234567\",\n  \"To\": \"+15557654321\",\n  \"Direction\": \"outbound\",\n  \"Status\": \"completed\",\n  \"StartTime\": \"2023-10-27T10:00:00Z\",\n  \"EndTime\": \"2023-10-27T10:01:30Z\",\n  \"Duration\": 90,\n  \"ErrorCode\": null,\n  \"ErrorMessage\": null,\n  \"ClientState\": \"user123\"\n}\n```\n\n### Python Example\n\n```python\nimport requests\n\ncall_id = \"call_12345abcde\"\nurl = f\"https://api.telegent.com/voice/calls/{call_id}\"\n\nheaders = {\n    \"Authorization\": \"Bearer YOUR_ACCESS_TOKEN\"\n}\n\ntry:\n    response = requests.get(url, headers=headers)\n    response.raise_for_status() # Raise an exception for bad status codes\n\n    call_details = response.json()\n    print(\"Call Details:\")\n    print(f\"Call ID: {call_details.get(\'CallId\')}\")\n    print(f\"Status: {call_details.get(\'Status\')}\")\n    print(f\"Direction: {call_details.get(\'Direction\')}\")\n    print(f\"Duration: {call_details.get(\'Duration\')} seconds\")\n\nexcept requests.exceptions.RequestException as e:\n    print(f\"Error retrieving call details: {e}\")\n    if response is not None:\n        print(f\"Response Body: {response.text}\")\n```\n\n### Call Details Retrieval Flow\n\n```mermaid\nsequenceDiagram\n    participant Y as Your Application\n    participant T as Telegent Platform\n\n    Y->>T: GET /voice/calls/{CallId} (Request Call Details)\n    activate T\n    T-->>Y: HTTP 200 OK (Call Details Response)\n    deactivate T\n```
