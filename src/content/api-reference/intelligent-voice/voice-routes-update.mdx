---\ntitle: Update Voice Route\n---\n\n## Update Voice Route\n\nThis endpoint allows you to update an existing voice route configuration.\n\n### Endpoint\n\n`/voice/routes/{RouteId}`\n\n**Method:** `PUT`\n\n### Path Parameters\n\n| Parameter | Type   | Description                         | Required |\n|-----------|--------|-------------------------------------|----------|\n| `RouteId`  | string | The unique identifier of the voice route to update. | Yes      |\n\n### Request Body\n\n| Parameter         | Type   | Description                                                              | Required |\n|-------------------|--------|--------------------------------------------------------------------------|----------|\n| `RouteName`       | string | Update the descriptive name for the voice route.                         | No       |\n| `InboundUrl`      | string | Update the URL for handling inbound calls (if RouteType is Webhook).     | No       |\n| `InboundUrlMethod`| string| Update the HTTP method for the `InboundUrl`.                             | No       |\n| `TransferNumber`  | string | Update the phone number to transfer calls to (if RouteType is Transfer). | No       |\n\nYou should include only the parameters you wish to update.\n\n```json\n{\n  \"RouteName\": \"Updated Inbound Call Webhook\",\n  \"InboundUrl\": \"https://your-app.com/updated-voice-webhook\"\n}\n```\n\nOR to update a Transfer route:\n\n```json\n{\n  \"TransferNumber\": \"+15551112222\"\n}\n```\n\n### Response Body (200 OK)\n\nA successful update request will typically return a `200 OK` response with the updated details of the route.\n\n```json\n{\n  \"RouteId\": \"VRID-abcxyz\",\n  \"RouteName\": \"Updated Inbound Call Webhook\",\n  \"RouteType\": \"Webhook\",\n  \"InboundUrl\": \"https://your-app.com/updated-voice-webhook\",\n  \"InboundUrlMethod\": \"POST\",\n  \"CreatedDate\": \"2023-11-02T10:00:00Z\" // Creation date remains the same\n}\n```\n\n### Response Properties\n\n| Property         | Type   | Description                                     |\n|------------------|--------|-------------------------------------------------|\n| `RouteId`        | string | Unique identifier for the voice route.          |      |\n| `RouteName`      | string | The updated name of the voice route.            |      |\
| `RouteType`      | string | The type of voice route (remains unchanged).    |      |\n| `InboundUrl`     | string | The updated URL for handling inbound calls.     | No   |\
| `InboundUrlMethod`| string| The updated HTTP method for the InboundUrl.     | No   |\
| `TransferNumber` | string | The updated phone number to transfer calls to.  | No   |\
| `CreatedDate`    | string | Timestamp when the route was created.           |      |\n\n### Python Example\n\n```python\nimport requests\n\nroute_id = \"VRID-abcxyz\"\nurl = f\"https://api.telegent.com/voice/routes/{route_id}\"\n\nheaders = {\n    \"Authorization\": \"Bearer YOUR_ACCESS_TOKEN\",\n    \"Content-Type\": \"application/json\"\n}\n\npayload = {\n  \"RouteName\": \"Customer Service IVR (Updated)\",\n  \"InboundUrl\": \"https://your-app.com/updated-ivr-webhook\"\n}\n\ntry:\n    response = requests.put(url, headers=headers, json=payload)\n    response.raise_for_status() # Raise an exception for bad status codes\n\n    updated_route = response.json()\n    print(\"Voice Route updated successfully:\")\n    print(f\"Route ID: {updated_route.get(\'RouteId\')}\")\n    print(f\"Updated Name: {updated_route.get(\'RouteName\')}\")\n    print(f\"Updated URL: {updated_route.get(\'InboundUrl\')}\")\n\nexcept requests.exceptions.RequestException as e:\n    print(f\"Error updating voice route: {e}\")\n    if response is not None:\n        print(f\"Response Body: {response.text}\")\n```\n\n### Update Voice Route Flow\n\n```mermaid\nsequenceDiagram\n    participant Y as Your Application\n    participant T as Telegent Platform\n\n    Y->>T: PUT /voice/routes/{RouteId} (Request Update Route)\n    activate T\n    T-->>Y: HTTP 200 OK (Updated Route Details)\n    deactivate T\n    T->>T: Apply Updates to Voice Route Configuration\n```
